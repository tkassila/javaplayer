JavaFx Audio, Video and Daisy Book Application

Purpose of the application

The application is playing audio or video files or a daisy book. A file or directory audio files at time. When selecting ncc.html daisy index file, the application is playing daisy voicebook after index web page, which is showed at lower part of the opened window.

Open a single audio or video file

Listen or see, pause, play again. Step previous or next 10 seconds backwards or further on audio files.

Open audio directory

By selecting an audio directory, you can listen all audio files from the beginning to the end. Go to previous or next file to play.

Possible preinstallation with Espeak screen reader users on daisy books

If you will choose the espeak screen reader and you have not installed it before, then it must to be installed before run this radiobutton with daisy books.

On windows, install setup_espeak-1.48.04.exe or a correspond setup (=installation) application. The javafxplayer application requires that espeek.exe etc is in the same directory to work!

Features

  • From start button
  • In daisy book are active: Previous or Next html link
  • In daisy book, you can browse or select on daisy index ncc.html on lower web component with mouse or tab and enter keys. When you have clicked mouse button or enter key pressed, the application is playing that mp3 file and voice position.
    "No TTS on daisy web page link focus" radiobutton is selected by default. All another radio buttons means, should a generated voice say selected web link text. When a screen reader user is pressing a tab key on open web ncc.html page and are coming on a new web page link. And before possible enter key press.
    With FreeTTS radiobutton selected, there are now installed only en-us language. You can install more language voices into FreeTTS, but perhaps there are any finnish language file to found.
    When Espeak radiobutton is selected, it requires preinstallation of the Espeak on Windows or Linux computer! See above Espeak header.

Key board commands

  • M: press M or m char = open music or audio play directory
  • F: press F or f char = open audio file
  • D: press D or d char = open daisy directory
  • P: press P char = pause music/voide or play music/voide
  • + or -: increase volume or decrease volume
  • 8 or 9: 10 sec backward or 10 sec forward of play position
  • 5 or 6: select previous or next audio file of audio directory
  • 2 or 3: select previous or next html link of daisy book index
  • Q or W: select previous or next level of daisy book index
  • R or T: select previous or next daisy link between selected index levels
  • Tab or Shift-Tab: select next/previous button,ui control or daisy web index page link. A tab key may used screen reader to say the text of an active ui control.
  • Enter: action like mouse click on active button etc or daisy web index page link

Gui controls

  • Open audio dir button: open selected directory, which contains audio files to play
  • Open audio or video file or daisy ncc index file -button: open one audio, viedo or daisy book file to play.

    When you have open daisy book, the daisy content is displayed at lower ui control. There are hyperlinks to browse or selected with a mouse, tab and enter keys.

  • From Start button: Move to start of file, or files
  • Previous file button: select previous file to play if a daisy book or directory has selected
  • Next file button: select next file to play if a daisy book or directory has selected.
  • Previous link button: select previous link to play if a daisy book has selected
  • Next link button: select next file to play if a daisy book has selected.
  • No TTS radio button: When daisy book has selected, and when a user will browse into daisy html link, should no TTS voice to say link text
  • Free TTS radio button: When daisy book has selected, and when a user will browse into daisy html link, Fee TTS voice synthesizer have to say link text with a voice. It has only english voice to speak.
  • Espeak TTS radio button: When daisy book has selected, and when a user will browse into daisy html link, Espeak TTS voice synthesizer have to say link text with a voice. This option requires a installation before this option.
  • Help button: Open this help dialog
  • BookMarks button: Control dialog for management of book marks
  • New BookMark button: Create new book mark
  • Play button: Play or Pause button when playing audio or video
  • 10 sec previous button: 10 seconds backwards when playing audio or video
  • 10 sec next button: 10 seconds forward when playing audio or video
  • Time slider: Time slider to position of the playing
  • Volume slider: Volume slider to control volume
  • Play time: Show current position
  • Meta information of file label: Show meta information of playing file
  • Message label: Show message of the application
